I must confess that I struggled with the Qt application. With this being my first project with the software, I spent a number of hours seeking understanding of the syntax. Apart from a strange phenomena in which the application causes my Linux system to move like a clumsy slug, I found the documentation difficult to parse. There was also a strange quirk in that I couldn't find documentation on the units used to express vram numbers in the /proc/#/stat file. Nonetheless, here is a modest and unfiltered result! I will consider spending more time with the application and Qt in the future, so as to better understand what is possible. I enjoyed the process (no pun intended), and I wouldn't mind working on things like this regularly, should I find the right resources for learning. For the moment, I take out of this class assignment, beneficial practice navigating the Linux file system in order to create such a display.
